# Iserlohn

Iserlohn is simple file uploader

## Installation

clone Github Repository

    $ git clone  https://github.com/naoto/iserlohn.git

Add Gem Library

    $ bundle install

## Usage

run file to bin/iserlohn

    $ bundle exec bin/iserlohn
